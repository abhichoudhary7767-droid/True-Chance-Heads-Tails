True Chance is a high-fidelity digital coin toss app designed to provide fair,
fast, and visually realistic random outcomes for everyday decisions.

It uses modern web technology to simulate a physical coin flip while ensuring
strong randomness and smooth animation.

---

## Features

- True 50/50 coin toss
- High-fidelity motion & physics-style animation
- Works offline
- No ads, no tracking, no accounts
- No external APIs or data collection

---

### Prerequisites
- Node.js 18+
